# BingMapTutorial

Android app that demonstrates work with Microsoft bing map
<br/>
### Features

- Support dark mode
- Show a map using Bing map.
- Show user location on the map.
- Implement a restful API call (Using Retrofit) to Google Places that get a list of nearby points of
Interests.
- Show the 5 nearest points of interests on the map and update them when the
location changes.

### Used technologies

-  100% Kotlin
-  Coroutines
-  Hilt (dependency injection)
-  Mvvm

### 3 party library

- Hilt
- Retrofit
- Dexter: for requesting permissions
- Bing Microsoft maps
